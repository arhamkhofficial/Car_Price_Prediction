# 🚗 Car Price Prediction

## 📌 Overview
This project predicts car prices using Machine Learning.

## 📊 Dataset
Car Price dataset from Kaggle.

## 🔍 Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training

## 🤖 Models Used
- Linear Regression
- Random Forest

## 📈 Results
Achieved good prediction accuracy on test data.

pip install -r requirements.txt
